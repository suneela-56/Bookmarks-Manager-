# 📑 Bookmarks Manager

A simple and user-friendly Bookmarks Manager built using **HTML**. This project allows users to organize and manage their favorite website links in a clean and structured interface.

## 🚀 Features

- Add and store bookmarks
- Organized bookmark listing
- Easy navigation to saved websites
- Clean and responsive user interface
- Beginner-friendly project structure

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
Bookmarks-Manager/
│
├── index.html
└── README.md
```

## 🎯 Purpose of the Project

This project was created to practice and improve HTML skills by building a simple web application that demonstrates webpage structure, forms, links, and layout organization.



## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/bookmarks-manager.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
